## 📥 Installation Guide

### Option 1: Installer (Recommended)

1. Download `VideoTool_Setup.exe`
2. Double-click to run
3. Choose installation location
4. Click "Install"
5. Launch from Start Menu or Desktop

### Option 2: Portable Version

1. Download `VideoTool_Portable.zip`
2. Extract to any folder
3. Run `VideoTool.exe`

> **Note:** No Python or FFmpeg installation required! Everything is bundled.
